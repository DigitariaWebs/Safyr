# Web Application

Next.js 16 App Router application (`@safyr/web`). Dark-first navy theme with cyan accent. Tailwind CSS 4, Radix UI, Zustand.

## Portals

- [[features/web/Public|Public]] — Marketing & landing pages
- [[features/web/Auth|Auth]] — Authentication flows
- [[features/web/Dashboard|Dashboard]] — Dashboard home & configuration

## Modules

- [[features/web/HR|HR]] — Human resources, employees, recruitment, training, discipline, legal registers, workflows
- [[features/web/Payroll|Payroll]] — Payroll calculation, social contributions, DSN declarations
- [[features/web/Billing|Billing]] — Invoices, quotes, purchase orders, VAT, client billing
- [[features/web/Accounting|Accounting]] — Journal entries, chart of accounts, bank reconciliation
- [[features/web/Banking|Banking]] — Bank accounts & transactions
- [[features/web/Planning|Planning]] — Sites, postes, shifts, schedule assignments, auto-scheduling
- [[features/web/Geolocation|Geolocation]] — Live tracking, presence zones, patrol rounds, reports
- [[features/web/Logbook|Logbook]] — Main courante, events, validation, security alerts
- [[features/web/Stock|Stock]] — Equipment inventory management
- [[features/web/OCR|OCR]] — Document scanning & extraction
