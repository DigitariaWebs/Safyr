# Documents (Mobile)

> Routes: `(app)/documents/index`, `(app)/documents/payroll`, `(app)/documents/schedule`
> Source: `mobile/src/app/(app)/documents/`

## Overview

Document hub for agents. Access payroll documents (payslips, tax forms) and schedule documents (assignment confirmations) from a single location.

## Workflows

### 1. Browse Documents
1. Agent opens documents screen
2. Two sections: Payroll Documents, Schedule Documents
3. Tap section → filtered document list
4. Tap document → view or download

### 2. Payroll Documents
1. List of payslips by period
2. Annual tax summaries
3. Download as PDF

### 3. Schedule Documents
1. Assignment confirmations
2. Planning exports

## Data Types

No dedicated types — aggregates from payroll and planning data.
