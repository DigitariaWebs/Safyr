# Home (Mobile)

> Route: `(tabs)/index`
> Source: `mobile/src/app/(app)/(tabs)/index.tsx`

## Overview

Dashboard tab — agent's home screen showing today's shift, quick stats, and shortcuts to key features.

## Workflows

### 1. Daily Overview
1. Agent sees today's assigned shift (site, times, poste)
2. Quick stats: hours worked this week, upcoming time-off, pending notifications
3. Shortcuts to: Geolocation, Main Courante, Schedule, Payroll

## Data Types

Aggregates from other features — no dedicated types.
